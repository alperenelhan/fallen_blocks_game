fallen_blocks_game
==================

Simple space invaders like game using interrupts

Installing
-------------

You need to compile GAME.C using Borland Turbo C/C++ 3.0.

Playing
-------------

`right_key,left_key:` Move hero to right, left

`Space:` Fire the gun

`Esc:` Quit game
